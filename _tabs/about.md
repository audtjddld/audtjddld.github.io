## Hi, I'm a developer based in South Korea.

I started my career writing Java and Spring-based backend services. Over time I transitioned to Kotlin, embracing its expressiveness and modern features for server-side development.

Recently, the rise of vibe coding has reshaped how I approach problems. I no longer limit myself to a single language or framework --- the right tool for the job is whatever gets the idea across clearly and quickly.

Everything I do still feels like a learning curve, and I'm fine with that. Challenging myself is what makes this work fun.

### Tech Stack

- **Languages:** Java, Kotlin
- **Frameworks:** Spring Boot, Spring Batch
- **CI/CD:** GitHub Actions, Docker
- **Infrastructure:** Kubernetes, Helm
- **Build:** Gradle, Maven
- **Others:** JPA, QueryDSL, Redis
