우분투에서 Eclipse 설치하는 방법을 소개 합니다.

Eclipse는 통합 개발환경을 제공합니다.

물론 무료구요.

소프트웨어 센터에서도 설치할 수 있지만

저는 다운로드 받아서 직접 설치 하는 방법을 선호 합니다.

(소프트 웨어 센터에서 설치 시 IDE를 따로 설치해야 함)



[우분투가 설치되지 않았다면 클릭](http://thinkubuntu.tistory.com/entry/%EC%9C%88%EB%8F%84%EC%9A%B0-8-%EC%9A%B0%EB%B6%84%ED%88%AC-%EB%A9%80%ED%8B%B0%EB%B6%80%ED%8A%B8-%EC%84%A4%EC%B9%98-%ED%95%98%EA%B8%B0)



**1. JDK 다운로드**

이클립스는 자바가 있어야 실행이 됩니다.

그래서 꼭 java가 깔려있어야 합니다.

SUN이 오라클에 인수된 후에 java는 많은 우여곡절이 있었습니다.

그래서 java는 몇 가지 종류로 나뉘었는데

오라클의 라이센스를 피하기 위해 OPEN Java가 있습니다.

그리고 오라클의 자바가 있습니다.



먼저 OPEN JDK설치는 간단합니다.

컨트롤+알트+T를 눌러 터미널을 여시고



$ sudo apt-get install openjdk-7-jdk



엔터 누르고 비밀번호 입력하면 됩니다.

간단하게 설치가 완료 됩니다.



**오라클 자바를 직접 다운로드 하여 설치할 수도 있습니다.**

저는 오라클 자바를 다운로드 받아서 설치 합니다.



<http://www.oracle.com/technetwork/java/javase/downloads/index.html>

![](http://cfile28.uf.tistory.com/image/0320383E51E490A028B03E)



![](http://cfile30.uf.tistory.com/image/033BD93E51E490A01F3F80)

라이센스 동의 체크하고 다운로드 합니다.

저는 우분트 13.04 amd64 환경이므로

Linux x64 를 다운로드 하였습니다.





2. 압축을 풉니다.**

그냥 노틸러스에서 압축 파일에 오른클릭 후 여기에 풀기

하셔도 되고 또는



$ cd ~/다운로드

$ tar zxvf jdk-7u25-linux-x64.tar.gz



압축파일을 풀면 jdk1.7.0_25 폴더안에 파일이 있게 됩니다.





**3. 파일을 이동시켜 줍니다.**



저는 이것을 /usr/lib/java 에 이동시키려고 합니다.

하기 전에 한 가지 확인 하고 갑니다.

터미널에서 다음 명령어를 쳐봅니다.



$ type java



무언가 발견되면 이미 자바가 설치되어 있거나 java라는 명령어가

이미 설정되어 있다는 뜻입니다.



![](http://cfile29.uf.tistory.com/image/0327984051E494351EE74C)

저는 없군요.

계속 설치를 진행합니다.

java 디렉토리가 있는지 확인합니다.

$ ls /usr/lib/java



없으면  java라는 디렉토리를 만들어야 합니다.



$ sudo mkdir /usr/lib/java



/usr/lib 디렉토리는 관리자 권한이 있어야 하기 때문에 sudo 명령어가 필요합니다.



파일을 이동시킵니다.



$ sudo mv ~/다운로드/jdk1.7.0_25 /usr/lib/java



파일이 잘 이동 되었는지 확인하고 다음을 진행합니다.





4. 환경변수 등록을 해줍니다.**

$ sudo gedit /etc/environment



gedit가 열리면

""안에 제일 끝에 " :  "후에 자바 설치 경로를 붙여 줍니다.

그리고 그 아래 줄에 JAVA_HOME을 설정해 줍니다.



PATH=".....:/usr/lib/java/jdk1.7.0_25"**JAVA_HOME=/usr/lib/java/jdk1.7.0_25



수정 하셧으면 저장 후 종료 합니다.



$ source /etc/environment



그리고 한번 실행시켜 설정을 적용 시킵니다.





5. 이제 링크를 걸어 주어야 합니다.**

이 작업은 터미널에서 java, javac, javaws 명령어를 사용할 수 있게 합니다.

만약 java명령어가 이미 다른 용도로 사용되고 있으면

java 1.7.0_25로 다시 설정해 줍니다.








$
sudo update-alternatives --install /usr/bin/java java
/usr/lib/java/jdk1.7.0_25/bin/java 1

$
sudo update-alternatives --install /usr/bin/javac javac
/usr/lib/java/jdk1.7.0_25/bin/javac 1

$
sudo update-alternatives --install /usr/bin/javaws javaws
/usr/lib/java/jdk1.7.0_25/bin/javaws 1 **



$
sudo update-alternatives --set java
/usr/lib/java/jdk1.7.0_25/bin/java

$
sudo update-alternatives --set javac
/usr/lib/java/jdk1.7.0_25/bin/javac

$
sudo update-alternatives --set javaws
/usr/lib/java/jdk1.7.0_25/bin/javaws



**6. 설치 확인 합니다.**

$ java -version



![](http://cfile25.uf.tistory.com/image/023F8B3E51E49BE70F5FDC)

자바 설치는 완료 되었고 이제 이클립스를 설치 합니다.

이클립스를 다운로드 받아서 설치 하겠습니다.





**7. 이클립스를 다운로드 받습니다.**



<http://www.eclipse.org/downloads/>



![](http://cfile25.uf.tistory.com/image/031B194551E49D710B9D8A)

이번에 juno에서 kepler로 바뀌었네요.

인디고나 주노를 더 선호하실분이 계실지 모르겠네요

저는 그냥 케플러로 갑니다.



![](http://cfile8.uf.tistory.com/image/21386D4551E49E1A37DCC2)





**8. 압축을 풉니다.**

마찬가지로 다운받은 압축파일 오른클릭 여기에 풀기

또는 터미널에서 명령어를 사용해도 됩니다.



$ tar zxvf ~/다운로드/eclipse-standard-kepler-R-linux-gtk-x86_64.tar.gz



이번에 작업 디렉토리를 안 옮겨서 내 폴더 안에 풀려 버렸네요.



$ cd ~/다운로드 해서 꼭 다운로드 폴더로 가서 압축해제 합시다.





**9. 파일을 이동시킵니다.**

압축을 풀면  eclipse 폴더가 있고 파일은 그 안에 모두 있습니다.

저는 eclipse를 /opt/에 설치 하겠습니다.



$ sudo mv ~/다운로드/eclipse /opt/



저는 내 폴더에 풀어서

**$ sudo mv ~/eclipse /opt/





10. 링크를 만들어 줍니다.**



자바는 링크를 걸어 줬지만

이클립스는 실행 스크립트를 만들어 줍니다.

만약 실행시 설정이 필요하다면

쉘 스크립트 파일을 수정해야 되므로 스크립트로 만들어 줍니다.



$ sudo gedit /usr/bin/eclipse



gedit 창이 열리면 아래와 같이 작성합니다.



----------------------------------------

#!/bin/sh**export
ECLIPSE_HOME=/opt/eclipse
$ECLIPSE_HOME/eclipse
$*

-----------------------------------------



![](http://cfile27.uf.tistory.com/image/221B644051E4A538273DC6)

저장 후 종료





11. 쉘 스크립트를 사용 가능하게 퍼미션을 설정해 줍니다.**

$ sudo chmod 755 /usr/bin/eclipse

모든 사용자에게 실행 읽기 권한을 줍니다.

![](http://cfile21.uf.tistory.com/image/0335023F51E4A60735B081)





12. 우분투 데스크탑 메뉴에 추가**



$ sudo gedit /usr/share/applications/eclipse.desktop



gedit 창이 열리면 아래와 같이 작성합니다..



----------------------------------------------------------------






[Desktop
Entry]

Encoding=UTF-8

Name=Eclipse

Comment=Eclipse
IDE

Exec=eclipse

Icon=/opt/eclipse/icon.xpm

Terminal=false

Type=Application

Categories=GNOME;Application;Development;

StartupNotify=true

----------------------------------------------------------------



![](http://cfile10.uf.tistory.com/image/226D2D4651E4A69113E093)



저장 후 종료 하면 설치가 완료 됩니다.

조금 설정 할께 많았지만 해볼 만 합니다.



![](http://cfile21.uf.tistory.com/image/02160D4151E4A7D7270DCC)



검색이 되면 아주 잘 설치 된겁니다.



![](http://cfile25.uf.tistory.com/image/2363E94151E4A8BF071A93)



잘 켜지는지 확인



![](http://cfile4.uf.tistory.com/image/2514F64151E4A8BF32F5B2)



런처에 등록시키려면 오른 클릭해서 런처에 고정 누르시면 됩니다.

이것으로 이클립스는 설치 완료 되었습니다.

클레식을 설치 하였기 때문에 현재 자바 개발환경은 만들어 졌습니다.











sudo chmod 755 -R eclipse



이클립스 설치폴더에 저렇게 선언하고 사용하면 된다.
