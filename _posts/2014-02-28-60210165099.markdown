JEUS 5 버전이 오래전에 개발되었기 때문에.

그때 당시 windows xp + ie 8 에서 한글로 파라미터를 넘기게 되면

엑셉트 랭귀지 타입이 ko로 넘어오는데 이때 jeus5 에서는 euc-kr로 인식하게되서

euc-kr로 인코딩을 실시하기 때문에 한글이 깨지게 된다.



이때는 tmax쪽으로 연락해서 해당 내용을 설명 후 패치를 하게되면 해결 할 수 있다.



ex) UTF-8로 인코딩해도 안된다.
