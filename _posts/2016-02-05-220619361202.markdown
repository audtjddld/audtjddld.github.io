1. Tomcat 설치하기**

$ sudo apt-get install tomcat6



**2. Tomcat 제거하기**

$ sudo apt-get remove tomcat6



**3. Tomcat 확인**

$ ps -ef | grep tomcat



**4. Tomcat 시작**

$ sudo service tomcat6 start

5. Tomcat 정지**

$ sudo service tomcat6 stop

6. Tomcat 설치경로**

/usr/share/tomcat6 (executable&libraries)

/var/lib/tomcat6 (conf&webapps)
