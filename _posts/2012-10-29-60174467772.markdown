```
function replaced(str){
            var data = str.replace(/>/gi,'>');
            data = data.replace(/</gi,'<');
            return data;
        }
```



/abc/gi, / /안에 글짜를 문자 끝까지 가면서 '>' 바꿔준다 뭐 이런 내용인듯.
