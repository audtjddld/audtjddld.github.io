상단에

<%@ taglib prefix="m" uri="/WEB-INF/tlds/method.tld" %>





tlds/method.tld 파일에



1**2
   <function>
        <description>makeMoney</description>
        <name>makeMoney</name>
        <function-class>com.module.trans.NumberConv</function-class> <!--클래스경로까지써주세요-->
        <function-signature>java.lang.String makeMoney(java.lang.String)</function-signature>
        <example>${m:makeMoney(str)}</example>
    </function>



기술한다



해당 내용은

NumberConv 라는 class 파일안에



makeMonery 라는 메소드를 호출 하는 것이다.




  public static String makeMoney(String num) {
    try {
      num = divideComma(num.trim(), 3);
    }
    catch (Exception e) {
    }
    return num;
  }
