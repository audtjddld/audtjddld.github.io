​

여기서 private method메서드를 호출해야 하니 invokeMethod 메서드를 사용해서 하겠다.

​









```
class Target {
 private String name;
 public Target(String name) {
  this.name = name;
 }

 private String addString(String value) {
   return name + value;
 }
}

class TestTarget {

  @Test
  void privateMethod() {
   Target target = new Target("ABC");
   String returnValue = ReflectionTestUtils﻿.invokeMethod(target, "addString", "DEF")
   org.junit.jupiter.api.Assertions.assertEquals("ABCDEF", returnValue);
  }
}
```
