​

|         application    |

​

**Git의 3대 자료구조**

​

**Blob**

​

Git commit 시 commit 될 때 파일 내용은 Hashing, 되어 Blob으로 저장 됨.

​

저장 위치

.git/objects

​

예를 들어

​

**Tree**

​

파일 A가 Hello world로 저장되어 있는데

파일 B도 Hello world로 저장하면

해시 값이 동일하다..

​

파일간의 관계를 Tree 형태로 저장한다.

​

​

**Commit**

작성자

Message

Parent commit

Add 된 파일들

​

40글자의 해시값으로 나온다.

SHA-1 로 저장하면 40 글자가 나온다.
