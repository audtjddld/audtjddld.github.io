하둡 설치 경로 (/home/junev/hadoop)



1. 첨부한 hadooop-eclipse-plugin-1.2.1.jar 파일을 자신의 이클립스 폴더의 plugins 폴더에 넣는다.

2. 이클립스를 실행하고, Window -> Open Perspective -> Other -> Map / Reduce 선택

![](/assets/images/posts/220120762500/a92059c9f158.jpg?type=w3)

3. 이클립스 하단 부에 Console 나오는 창에 보면 Map / Reduce Location 탭을 선택하고 오른쪽 마우스 클릭 후 , New Hadoop location을 선택한다.

![](/assets/images/posts/220120762500/acf6409a0d47.jpg?type=w3)



4.
 Location name 은 아무거나 작성하고, Map/Reduce Master 의 호스트와 포트는 자신이 설치한 하둡의
conf 폴더에 core-site.xml 에서 설정한대로 적어주고, DFS Master의 포트는 mapred-site.xml에서
설정한대로 적어준다.



![](/assets/images/posts/220120762500/ab6406f0bef4.jpg?type=w3)



5. 위의 과정이 끝났다면 이클립스 좌측부에 다음과 같이 창이 생긴다.

![](/assets/images/posts/220120762500/cb2acf459dd1.jpg?type=w3)



만약 모든 폴더 및 파일이 열리지 않고 Permission Denied 에러가 뜬다면,  $HADOOP_HOME/conf/hdfs-site.xml 에서 다음과 같이 프로퍼티를 추가한다.

<property>

<name>dfs.permissions</name>

<value>false</name>

</property>



 6. 이제 하둡 맵 리듀스 프로젝트를 만들어보자. 이클립스에서 File -> New -> Other -> Map / Reduce Project 선택

![](/assets/images/posts/220120762500/ea1270e0a082.jpg?type=w3)

프로젝트이름을 정하고, Configure HAdoop install directory를 선택하여 자신이 설치한 하둡의 디렉토리를 선택한다.





![](/assets/images/posts/220120762500/8cde8ea85be0.jpg?type=w3)**7. 무사히 프로젝트를 만들었다면 간단한 wordcount를 만들어본다.
