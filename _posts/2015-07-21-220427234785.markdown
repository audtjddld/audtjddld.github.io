메시업은 팀 오라일리 (Tim O' reilly)의 '웹 2.0이란 무엇인가?' 라는 글에서 명시적으로 제시되어있는데, 해당 글에서는 리믹스 가능한

'리믹스 가능한 데이터 소스' 그리고 '리믹스 할 수 있는' 이라는 형식으로 묘사되었다. 팀 오라일리가 구축한 통합적인 웹 개념의 토대와 이러한 개념을 미리 인식하고 있던 엔지니어들은 기존의 기술과 신기술을 사용하여 '웹 2.0 기술'이라는 것을 구현한다. 데이터들의 연동을 웹 서비스 그리고 애플리케이션 간의 리믹스로 새로운 개념의 실시간 웹 서비스를 이용하여 데이터를 제공하는 이 웹 2.0의 마인드셋은 사용자와 고객에게 더욱 그들이 원하는 것을 빠르고 혁신적으로 접합 할 수 있는 기회의 장을 열어줬다.

매시업 기술이 더욱 번차하게 된 이유는

1. XML 웹 서비스의 사용

2. RESTFul 웹 서비스의 사용

3. ajax의 본격적인 사용

4.javascript 의 다양한 활용과 다양한 MVC 라이브러리의 탄생

5. 데이터 리믹스와 매시업에 대한 관심

6. 클라우드 서비스의 진화

7. 웹 브라우저의 끊임없는 진화 (Google Chrome, Firefox, Safari)

8. LTE, LTE-Advaced, Super wifi 등과 같은 초고속 인터넷

9. 뛰어난 UCC, UGC의 보급

10. 모바일 시대의 개막과 대중적인 스마트폰의 보급

11. 빅 데이터를 분석하는 서버 성능의 확대와 진화

등이다.
